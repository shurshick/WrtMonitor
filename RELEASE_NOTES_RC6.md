# WrtMonitor v0.1.1-rc6

## Автообновление OpenWrt agent

- Агент автоматически проверяет обновление на собственном сервере WrtMonitor при запуске и далее раз в шесть часов.
- Источник обновления: `<адрес_сервера>/downloads/openwrt/`; GitHub для этого не требуется.
- Перед установкой скачанные shell-скрипты проходят проверку `sh -n`.
- UCI-конфигурация, `device_id`, `device_token` и имя роутера не изменяются.
- Автообновление включено по умолчанию. Для отключения: `uci set wrtmonitor.main.auto_update='0'` и `uci commit wrtmonitor`.
- Для немедленной проверки доступна команда `wrtmonitor-agent update`.
