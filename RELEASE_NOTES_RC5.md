# WrtMonitor v0.1.1-rc5

## Управление устройством

- Добавлено отключение роутера из Web UI и Android.
- Сервер отправляет агенту команду отключения. После подтверждения агент выключает себя и прекращает опрос до повторного подключения.
- Отключённое устройство сохраняется в истории со статусом `disabled`; telemetry и команды не удаляются.

## Wi-Fi

- Добавлена смена пароля Wi-Fi в Web UI и Android.
- Минимальная длина нового пароля: 8 символов.
- Пароль маскируется в API/истории команд и очищается из очереди после ответа агента.

## Интерфейсы

- Android: кнопки Wi-Fi, сети и системы приведены к единой полной ширине без переносов.
- Время последней связи, обновления telemetry и uptime отображаются в читаемом виде.
- В Web UI добавлены отступы между элементами управления Wi-Fi и сети.

## Обновление

Для функций отключения и смены пароля требуется обновить и сервер, и OpenWrt agent до `v0.1.1-rc5`.
