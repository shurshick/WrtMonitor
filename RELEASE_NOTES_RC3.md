# v0.1.1-rc3-session-telemetry

## Исправления

- Android теперь распознаёт `HTTP 401` как истечение сессии, очищает устаревший токен и возвращает на экран входа вместо остановки мониторинга.
- Проверка обновлений использует список GitHub releases и поддерживает prerelease.
- `versionCode` повышен до 19: APK устанавливается поверх rc2.

## Телеметрия

- OpenWrt agent передаёт CPU, место на overlay/корневом накопителе, статусы интерфейсов, данные радиомодулей и расширенные ответы ubus.
- Экран устройства показывает свежесть данных, аптайм, нагрузку, память, CPU, накопитель, оборудование, сеть и Wi-Fi отдельными блоками.

## Артефакты

- TrueNAS YAML;
- архив OpenWrt agent;
- Android debug APK;
- `SHA256SUMS.txt`.
